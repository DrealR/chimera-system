## Camping = Extraction (Immune Response)

```
WHEN SOMEONE CAMPS AT A SPAWN:

  They sit on a resource point.
  People keep appearing.
  They keep extracting (kills).
  Seems like good strategy.

BUT:

  1. Everyone learns where they are.
     Predictable = vulnerable.

  2. They're not PLAYING the game.
     They're damming the flow.

  3. Other players adapt around them.
     System routes flow elsewhere.

  4. SPAWNS FLIP.
     The game's IMMUNE SYSTEM responds.

SPAWN FLIPPING = the game
REROUTING FLOW
away from the extractor.

When a camper sits at A spawn,
the game starts spawning
people at B spawn instead.
The flow ABANDONS the camper.

THE GAME HAS AN IMMUNE SYSTEM
AGAINST EXTRACTION.

SAME AS:
  Monopoly → customers find
  alternatives → monopoly dies.

  Dam → water finds new path →
  dam becomes irrelevant.

  Cancer → body reroutes blood
  around tumor if it wins.

CAMPING = MONOPOLY = DAM = CANCER.
All the same topology.
All trigger immune response.
All eventually abandoned by flow.
```
